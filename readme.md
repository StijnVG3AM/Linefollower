## specifications
<br />
microcontroller: ATmega328 5V 16MHz
<br />
motors: Gear motor 6V - 50:1
<br />
h-bridge: DRV8833
<br />
sensors: QTR-8A
<br />
batteries: 2x Quickstuff 18650 batterijen
<br />
wireless communication: HC-05 bluetooth module
<br />
distance sensor - motors: 12cm
<br />
weight: 238g
<br />
speed: 0,446 m/s
<br />
